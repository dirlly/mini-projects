Modern Clock

This is a mini project exploring a modern digital clock built with an ESP32. Inspired by Huy Vector, this build prioritizes simplicity and the honest display of electronic components.

Features

  • - Precision Timekeeping
  • - Temperature Monitoring
  • - Day & Date Teller
  • - Customizable Alarm
